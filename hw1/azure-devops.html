Azure DevOps

Microsoft's platform combining work planning, source control, build and release automation, test management, and package hosting in a single integrated service.

Azure DevOps is a cloud-hosted platform covering every stage of the SDLC. It integrates with Azure, the Power Platform, and GitHub Azure Pipelines can build and deploy directly from GitHub repositories.

  • Azure Boards — Agile work tracking with Scrum and Kanban boards, epics, user stories, and sprint planning
  • Azure Repos — unlimited private Git repositories with branch policies and required pull request reviews
  • Azure Pipelines — CI/CD automation using YAML-based multi-stage pipelines
  • Azure Test Plans — manual, exploratory, and automated test management linked directly to work items
  • Azure Artifacts — universal package feeds for NuGet, npm, Maven, and Python packages
  • Pipeline as Code — storing the YAML pipeline in the same repo means changes are versioned and reviewed like any other code
  • Environment approvals — required sign-offs gate deployments between Dev → Test → Production stages
